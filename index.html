Услуги инженера-конструктора

Предлагаю услуги проектирования и расчётов в машиностроении и строительстве, с акцентом на точность, скорость и высокое качество:

1. Расчёты сосудов и аппаратов:

- проверочные расчёты емкостного оборудования, сосудов и аппаратов под давлением / без давления различных типов с использованием программы «Пассат v1.09R2» НТП «Трубопровод».

- создание детализированных 3D-моделей и чертежей в САПР Компас 3D ПО АСКОН;

- разработка чертежей марки КМД на основе проектов КМ;

- разработка рабочей документации на основе проектов АР.

3. Исполнительная документация:

- подготовка исполнительной документации для сдачи смонтированного оборудования и трубопроводной обвязки на объектах нефтегазовой отрасли.

4. Студенческие работы:

- выполнение курсовых, рефератов и других работ по инженерным дисциплинам с учётом всех требований.

Возможности AI версии (в дополнение к базовым):

  • 🧠 Интеллектуальный анализ 3D-модели — ИИ оценивает состав, готовность документации, выявляет проблемные компоненты
  • 📋 Автозаполнение основной надписи по данным модели, или на основе требований пользователя — ИИ предлагает значения полей (организация, разработчик, проверяющий и т. д.)
  • 💬 Чат-ассистент по ГОСТ, КОМПАС-3D и работе с утилитой (отвечает на любые технические (и не только) вопросы)
  • 📜 Долговременная память чата — запоминает факты об организации, проекте, сотрудниках - и использует эту информацию при анализе проекта
  • 📊 HTML-отчёт с детализацией по каждому чертежу
  • 🔧 Профили основной надписи — сохранение реквизитов разных организаций или проектов
  • ⚙️ Гибкие настройки ИИ: выбор AI-модели из списка, или ручной ввод любой доступной современной

Базовые возможности (как в Free версии):

  • 4 вида: Спереди, Слева, Сверху, Изометрия
  • Автоподбор формата листа от А4 до А0
  • Автоподбор масштаба по ГОСТ 2.302-68
  • Автоматическая простановка габаритных размеров
  • Заполнение основной надписи по ГОСТ 2.104
  • Визуальный редактор штампа как в КОМПАС
  • Цветовая подсветка компонентов по типу
  • Фильтрация: сборочные единицы, детали, стандартные изделия и т. д.

Системные требования:

  • ОС: Windows 10/11 (64-бит)
  • КОМПАС-3D: v19 или выше
  • Память: 8 GB RAM (рекомендуется не менее 16 GB для локальных ИИ-моделей)
  • Место на диске: 90 MB утилита + 5,2 GB Ollama + место для моделей Ollama (облачные почти не занимают места, для локальных потребуется 1–6 ГБ)
  • Интернет: для загрузки моделей и использования облачных ИИ

Как использовать AI версию:

  1. Установите Ollama (инструкция в настройках ИИ) и загрузите модель (например, рекомендуемую glm-5:cloud)
  2. Запустите KompasAutoDrawAI
  3. Откройте 3D модель в КОМПАС-3D
  4. Нажмите "Загрузить из активного документа"
  5. Используйте ИИ-функции: анализ модели, заполнение штампа, чат
  6. Нажмите "Экспортировать выбранные в CDW"

Пробная версия: полный функционал без ограничений в течение нескольких дней. Для приобретения постоянной лицензии обратитесь к разработчику.
